Realizamos una página web utilizando HTML, CSS, SweetAlert y Bootstrap. El sitio consta de tres secciones. La primera sección es una presentación del trabajo. La segunda sección contiene un botón que cuando se lo presiona consulta una API provista por el docente. La API devuelve la latitud y la longitud que carga un marcador en el mapa de la sección 3. La tercera y ultima sección contiene de un mapa. Este mapa es una herramienta de HERE Technologies, que provee de diferentes herramientas y recursos para la localización de coordenadas
